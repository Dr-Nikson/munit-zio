# munit-zio

Integration library between [MUnit](https://scalameta.org/munit) and
[ZIO](https://zio.dev).

### Goals

+ Suite
+ Assertions
- Resource Fixtures
+ Clear logs
- Layers providing
- Custom Errors handling
- Scala Steward
- Examples in README

Inspired by
[munit-cats-effect](https://github.com/typelevel/munit-cats-effect).
